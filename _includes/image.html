{% comment %} Pouziti: `include image.html [img={string}] src={string} [href={string}] alt={string} title={string} [caption={string|markdown}] [attributes={string}]` - `img`: Stejně jako u `code.html` umožňuje zadání obrázku pomocí markdown zápisu `![]()`. - `src`, `href`: jsou cesty k obrazku (obrazkum nahled/plny) - `alt`: popis obsahu obrazku - `title`: titulek obrazku - `caption`: popisek (jeden paragraf) - `attributes`: HTML atributy pro `` (primárně výška/šířka).
